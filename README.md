# riumr GitHub Portfolio

`index.html` の内容に合わせた、GitHubプロジェクト紹介ページです。  
学習と実装を継続しながら、Web開発の基礎から実践までを積み上げている開発者として、現在 Pin している主要リポジトリを掲載しています。

## 掲載プロジェクト

1. [complaintsAnaysis](https://github.com/riumr/complaintsAnaysis)  
   苦情データの傾向把握とインサイト抽出を目的とした分析プロジェクト。
2. [Java-chip](https://github.com/riumr/Java-chip)  
   Django と Bootstrap で構築したコーヒー豆販売サイト。
3. [teamproject-frontend](https://github.com/riumr/teamproject-frontend)  
   チーム開発でフロントエンド実装を担当したプロジェクト。
4. [TIL](https://github.com/riumr/TIL)  
   Django・JavaScript・CS を中心に学習内容を記録しているリポジトリ。
5. [project-backend (devman26173)](https://github.com/devman26173/project-backend)  
   Spring Boot / Thymeleaf ベースのコミュニティサービス「J-BOARD」。
